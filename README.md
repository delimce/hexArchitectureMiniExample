# Hexagonal Architecture mini example using PHP programing language

## Prerequisites
- Docker engine
- Mysql >= 5.7

## Installation
    1. git clone git@github.com:delimce/hexArchitectureMiniExample.git
    2. docker-compose up
    3. edit /etc/hosts file adding new entry -> 127.0.0.1   dev.telecoming.es
    4. composer install
